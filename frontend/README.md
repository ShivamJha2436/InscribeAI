## Frontend

### Stack
- Next.js + TypeScript
- Zustand for state (`/lib/store/*`)
- Zod for validation
- Tailwind for styling

### Getting Started
```bash
npm install
npm run dev
```

Open http://localhost:3000

### Auth
- Login/Register pages use Zod validation and call backend at :8080

### Docs
- See `/docs` route for product docs

### Typography
- Helpers in `lib/utils.ts` (`typography` export)
